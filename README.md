# 🌦️ Weather Dashboard

A responsive Weather Dashboard built using **HTML, CSS, JavaScript, and Bootstrap**. The application fetches real-time weather data from a Weather API and provides a clean, user-friendly interface for viewing current weather conditions and forecasts.

## ✨ Features

* 🔍 Search weather by city
* 📍 Current location weather
* 🌡️ Real-time temperature and weather conditions
* 💧 Humidity
* 💨 Wind speed
* 🌅 Sunrise & Sunset
* ☀️ UV Index
* 🌫️ Air Quality Index (AQI)
* 📅 Multi-day weather forecast
* 📱 Fully responsive design

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Weather API
